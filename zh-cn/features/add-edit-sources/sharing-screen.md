# 🖥️ 共享电脑桌面或应用

在文档内容区点击【+】号，在打开的窗口中选择【共享屏幕】，然后选择要共享的应用或者桌面即可。

<figure><img src="../../.gitbook/assets/image (50).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
注意：共享应用时，不能将应用最小化，最小化之后共享中断。
{% endhint %}
