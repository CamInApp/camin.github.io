# 📳 手机/平板投屏到 CamIn

在文档内容区点击【+】号，在打开的窗口中选择【投屏】，你可以选择使用苹果投屏或 ClassIn 投屏。根据窗口中的提示操作即可。

{% hint style="info" %}
请确保投屏设备和电脑连接到相同网络。如果发现无法投屏时，请尝试按照这个方法解决👉 [tou-ping-shi-bai.md](../../faq/tou-ping-shi-bai.md "mention")
{% endhint %}
